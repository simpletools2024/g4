# Building

## Needed to host:
- Python 3.x installed
- Terminal
- Visual Studio Code

## How do i do this?
1. Clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Run the Python HTTP server:

```bash
python -m http.server 8000
```
Once done you'll have Granny self hosted!
